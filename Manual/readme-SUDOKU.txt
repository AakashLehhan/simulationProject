How to play?
	1. Run the source code. (DO NOT MODIFY THE CODE TO AVOID ERRORS) 
	2. When the resulting window will appear:
		a. Fill the board (Follow the classic Sudoku rules)
		b. Press the check button to check the errors.
			Errors:
				Red: More than a character or you provided a space or you haven't gave any input.
				Yellow: Possibility of similar characters in a row or column or in a cube.
	3. Press the "Check" button to submit the response and see your result in the popup.
	4. Press the "Reset" button to reset the board and start over.
